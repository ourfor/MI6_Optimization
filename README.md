> 本指南将在官方MIUI的系统基础上对MI 6进行个性化，以达到稳定省电的目的。
> 一个稳定的系统可以极大的提高学习的效率       ---- ***鲁迅***

So,不修改系统才能保证原来的稳定，所以这里使用Magisk模块(一部分自制，一部分来自酷友)：

# 好用的Magisk模块
- 时间居中模块 miui_center_clocker
- MIUI主题（整合安卓7.0开机画面）
- 全面屏手势（包含禁用系统升级，Google相机代码）
- mm管理器（方便recovery下面卸载模块）
